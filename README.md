# Databricks Training Repository

This repository contains SQL and PySpark exercises for the Databricks training workflow, organized by phase with notes, output tables, and output images.

## Repository structure
- Phase 1: filtering and selection exercises
- Phase 2: joins and aggregation analysis
- Phase 3: ingestion and cleaning pipeline
- Phase 4a: bucketing and segmentation
- Phase 4b: data cleaning workflow
- PySpark Assignments: practice notebooks and datasets
- Datasets-PySpark Project: source CSV files

## Included artifacts
- SQL query files
- PySpark notebook and script files
- Output tables in markdown
- Output images in SVG format
- Phase-specific README notes

- Phase 2: Added joins and aggregation queries
